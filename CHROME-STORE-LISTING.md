## Detailed Description

🎨 **CSS Copy Tool** — The fastest way to copy CSS from any website element.

Stop manually inspecting elements in DevTools. CSS Copy Tool lets you hover over any element and instantly copy its computed styles in the format you need.

**How it works:**
1. Click the extension icon or press **Alt + Shift + C**
2. Hover over any element — it highlights instantly
3. Click to see all computed CSS properties
4. Copy everything, or select just the properties you need

**🚀 Key Features:**

• **One-Click CSS Copying** — Hover, click, copy. That's it.
• **3 Output Formats** — Pure CSS, Tailwind classes, or inline styles
• **Smart Property Categories** — Colors, typography, spacing, borders, layout, and effects, organized and filterable
• **Selective Copy** — Copy all properties or pick individual ones
• **Keyboard Shortcut** — Alt + Shift + C to toggle on/off instantly
• **Element Info** — See tag name, ID, and class of the selected element
• **Color Previews** — Visual color swatches next to color values
• **Lightweight** — Under 50KB total. No bloat, no frameworks.
• **Works Everywhere** — Any website, any element

**🎯 Perfect For:**

• Frontend developers recreating designs from existing sites
• Designers who want exact CSS values without DevTools
• Tailwind CSS users who need quick class conversion
• Anyone tired of right-click → Inspect → scroll through computed styles

**💰 Pricing:**

• **Free** — Full CSS copying with all core features
• **Pro ($19 one-time)** — Tailwind export, advanced property filtering, priority support

**🔒 Privacy First:**

This extension does NOT collect, store, or transmit any data. Everything runs locally in your browser. No accounts, no analytics, no tracking. Your browsing stays private.
